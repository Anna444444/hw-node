Курс: Создание приложений на Node.js
Тема: Введение. Модули

Задание 1
=========
Зарегистрируйте акканунт на github.com и создайте репозиторий для домашних работ по курсу Node.js.
Склонируйте этот репозиторий на свой домашний компьютер с помощью программы Github Desktop.
Внутри этого репозитория создавайте папки для каждой домашней работы, например, "Jun-05" и все задачи из домашней работы выполняйте в ней.
По выполнению заданий делайте фиксацию изменений и синхронизацию с сервером: Commit и Push.
На MyStat вместо архива с вашими решениями, предпочтительно отправлять ссылку на ваш github репозиторий.

Задание 2
=========
Установите на ваш компьютер node.js. Дистрибутив можно загрузить с официального сайта nodejs.org

Задание 3
=========
Создайте новый проект (npm init), в котором:
 - создадите модуль в стиле Common JS, повторяющий модуль cowsay, но который в терминале рисует другое животное - на ваш выбор;
 - в главном файле проекта подключите свой модуль, используя функцию require() и продемострируете работу модуля.

Задание 4
=========
Создайте новый проект (npm init), в котором:
 - создадите ES6 модуль, содержащий функции для вычисления площади и периметра разных геометрических фигур - трёх, на ваш выбор;
 - в главном файле проекта подключите свой модуль, используя инструкцию import и продемострируете работу модуля.
