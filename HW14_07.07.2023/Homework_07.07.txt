Курс: Создание приложений на Node.js
Тема: Socket.io


Задание 1
=========
С использованием socket сделать страницу со следующими возможностями:
    1. При новом подключении добавлять socket.id в массив и выводить на странице id всех подключений
    2. По нажатию на id отправлять конкретному подключению любое сообщение. Сообщение должно выводиться в поле для сообщений как приватное.
    3. При отключении – удалять id из массива и обновлять список id на странице
